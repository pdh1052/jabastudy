클래스 변수명 = new 생성자();
1. 생성자의 역할:
   -멤버변수의 초기화
   -객체(=참조자료형의 변수) 생성 때, 자료형에 맞는 값으로 사용.

2. 멤버 변수의 역할:
   -객체의 구성 요소를 표현함.
   -메서드에서 중복되는 값을 표현함

3. 지역변수:
   -메서드 안에서 중복되는 값을 표현함.

4. 메서드의 역할
   -객체의 기능(=능력)을 표현.
   -클래스 안에서 중복되는 값들을 표현함.

5. 디폴트 생성자는
   - 다른 생성자가 눈에 안보이면 존재한다.
   - 다른 생성자가 눈에 보이는데, 디폴트생성자가 안보이면 존재하지않는다.

6. this : 객체 자기 자신.
   -this를 왜 붙이느냐? 지역변수와 멤버변수가 이름이 같으면
                               컴파일러는 지역변수로 인식한다. 만약에
                               멤버변수를 가리키고 싶다면, >> this.멤버변수 

======객체(Object or instance or bean or entity or 도메인)=======

1.객체는 국어사전의 명사다.
   - 자동차 >>>  Car bmw = new Car();
   - 강아지 >>> Dog happy = new Dog();
2. heap 메모리에 저장된 데이터.
3.실체
  
객체 생성법
참조자료형 변수명(=객체명=레퍼런스) = new 생성자();
객체가 갖고 있는 자원(변수, 메서드)을 사용하는 방법
>>>객체명.변수명
>>>객체명.메서드명();
